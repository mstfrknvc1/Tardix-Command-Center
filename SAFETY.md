# Safety Guide

## Before Using
- Confirm your model is listed as supported, or test carefully in limited mode.
- Ensure cooling is functional and vents are clean.
- Avoid stress workloads while first testing power/fan changes.

## Safe First Run
- Start with default power profile (Balanced).
- Verify temperature and fan readings before applying manual fan boosts.
- Test RGB separately from power/fan settings.

## If Something Looks Wrong
- Revert to default settings in app.
- Disable autostart or any custom background service you enabled.
- Reboot and verify stock behavior.
- Run uninstall.sh to remove integration files.

## Thermal Responsibility
Always monitor CPU/GPU temperatures. Do not rely on this tool as the only safety control.
